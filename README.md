## Run Locally

Clone the project

```bash
  git clone https://github.com/nitinduseja/laravel8-admin-panel
```

Go to the project directory

```bash
  cd laravel8-admin-panel
```

Install dependencies

```bash
  composer install
```

Laravel Migrate

```bash
  php artisan migrate --seed
```

Start the server

```bash
  php artisan serve
```
